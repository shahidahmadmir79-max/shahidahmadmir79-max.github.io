## 🚀 Getting Started

### Run locally

1. Clone the repo:
   ```bash
   git clone https://github.com/your-username/marketing-by-shahid.git
   ```
2. Open `index.html` in your browser — that's it.

### Deploy to GitHub Pages

1. Push all files to a GitHub repository
2. Go to **Settings → Pages**
3. Set source to `main` branch, root folder
4. Your site will be live at `https://your-username.github.io/marketing-by-shahid`

### Deploy to Netlify (recommended)

1. Drag and drop the project folder at [netlify.com/drop](https://netlify.com/drop)
2. Live instantly — no account needed for a quick test

## 📬 Connect the Contact Form

The form is currently static. To make it send emails:

1. Go to [formspree.io](https://formspree.io) and create a free account
2. Create a new form and copy your form ID
3. In `index.html`, find the `<form>` tag and add:
   ```html
   <form action="https://formspree.io/f/YOUR_FORM_ID" method="POST">
   ```
